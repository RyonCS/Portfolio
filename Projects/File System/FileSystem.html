File System

C - Language

For this project, I used C to develop a server and client capable of communicating and transmitting files.

The client connects to the server via socket programming and uses command line arguments to send, retrieve, or delete files on the server.

The server is responsible for storing files and maintaining a version history. When the server receives a write command from the client, it stores the file in the specified directory. If a file with the same name already exists, a copy is made, and the version history is updated accordingly.
