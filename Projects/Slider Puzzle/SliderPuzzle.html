Slider Puzzle

Northeastern University, Fall 2022
Python, OOP

This is the first project I created as part of my Computer Science program. I utilized Python and taught myself Object-Oriented Design to create this game.

The game loads in an initial puzzle and allows the user to click on an image tile adjacent to the empty tile to shift the tiles around until the puzzle is complete. The user can click on reset to solve the puzzle, can click on load to load in a new puzzle, or quit the game. After the user successfully solves the puzzle, their data is stored in a file and displayed in the leaderboard section.

Some slider puzzle configurations are not solvable, so for quality-of-life, I implemented an algorithm to check if the puzzle is solvable or not. If the puzzle is not solvable, a new version of the puzzle is loaded automatically. This implementation keeps the user from endlessly trying to solve an unsolvable puzzle.
